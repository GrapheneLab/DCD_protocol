## Description
Retrieve accounts which are servants of a given account 

## Info

**Command**

```sh
clsig get servants
```
**Output**

```console
Usage: clsig get servants account

Positionals:
  account TEXT                The name of the controlling account
```

## Command

```sh
clsig get servants inita
```

## Output

```json
{
  "controlled_accounts": [
    "tester"
  ]
}
```
